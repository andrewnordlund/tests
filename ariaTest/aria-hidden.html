Summary

When using aria-hidden="true", keep the following in mind:

  • Adding aria-hidden="true" to a non-focusable element is well supported in screen reader and browser combos.
  • But adding tabindex="0" has varying results:
    • Firefox (55) on OSX (10.9), and IE 11 with both NVDA and JAWS on Windows 7, aria-hidden="true" is ignored completely.
    • JAWS + Opera reads the whole <div>.
    • NVDA + Firefox (55 - 60) messes up code in surrounding paragraphs.
